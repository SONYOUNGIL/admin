# ONE-DEV
### ONE 프로젝트


# 설명
- ### **Kubernetes**
    #### Kubernetes 매니페스트 파일이 위치한 폴더 입니다.

- ### **.gitlab-ci**
    #### 깃랩 CI를 위한 스크립트 파일 입니다.
- ### **cloudbuild**
    #### Google Cloud Build 스크립트 파일 입니다.
   빌드 순서 및 내용이 정의 되어 있습니다.
- ### **skaffold**
    #### Google Cloud Deploy 스크립트 파일 입니다.
- ### **clouddeploy**
    #### Google Cloud Deploy 배포 파이프라인 스크립트 파일 입니다.
    최초 한번 실행 후 재실행 필요가 없습니다.
